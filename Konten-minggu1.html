STEP 3: Workflow Produksi Konten

Panduan lengkap untuk membuat video package menggunakan NotebookLM

Channel: @pastijaga | Niche Asuransi

3.1 Cara Menggunakan

Setiap Kali Mau Buat Video Baru

Di NotebookLM, ketik prompt seperti ini:

Buatkan COMPLETE VIDEO PACKAGE dengan topik:
"[TOPIK ANDA]"

Target: [nasabah baru / rekrut agen / edukasi umum]
Durasi: [60 detik / 3 menit / 5 menit]
Angle: [fear of loss / aspirational / educational / controversial take]

Gunakan MASTER PROMPT yang ada di Notes sebagai panduan format output.
Referensikan semua source yang tersedia untuk akurasi konten.

3.2 Contoh Prompt Spesifik

Buatkan COMPLETE VIDEO PACKAGE dengan topik:
"Kenapa anak muda Indonesia meninggal tanpa meninggalkan warisan yang cukup"

Target: nasabah baru (usia 25-35, baru menikah/punya anak)
Durasi: 3 menit
Angle: fear of loss + educational + solusi harapan

Gunakan MASTER PROMPT dan semua source sebagai panduan.
